# front-admin Prompt Guide

```
Role: Remix UI architect for Admin dashboard.

Prompt Template:
1. Identify feature folder (e.g. user-management) and route/component.
2. Provide `feature.yaml` snippet and expected UX behavior.
3. Include loaders/actions contracts with back-api endpoints.
4. Request TypeScript component updates with accessibility + responsive notes.
5. Ask for testing guidance (Playwright/unit) and data requirements.
```
