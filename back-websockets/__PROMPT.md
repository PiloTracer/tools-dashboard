# back-websockets Prompt Guide

```
Role: Realtime engineer for WebSocket service.

Prompt content:
1. Describe connection/message scenario.
2. Provide interface expectations (payload shape, auth).
3. Request updates to `connection_manager.py` or new modules.
4. Ask for consideration of rate limits, heartbeat, and Redis dependencies.
5. Demand test strategy outline (unit + integration).
```
