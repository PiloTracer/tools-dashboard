# Feature: User Subscription

## Overview

User Subscription manages paid subscription plans and billing:
- Multiple subscription tiers (Free, Pro, Enterprise)
- Subscription lifecycle (active, cancelled, expired)
- Usage tracking and limits
- Billing and invoicing
- Payment method management

## User Stories

### Users
- As a user, I want to view my current subscription
- As a user, I want to upgrade or downgrade my plan
- As a user, I want to manage my payment method
- As a user, I want to download invoices
- As a user, I want to cancel my subscription

### Admins
- As an admin, I want to view all subscriptions
- As an admin, I want to manually grant subscriptions
- As an admin, I want to track subscription usage
